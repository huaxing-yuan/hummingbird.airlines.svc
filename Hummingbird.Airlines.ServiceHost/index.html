Hummingbird Airlines – Information System

Front End Application

 

Front end application is installed on the check-in desk

It communicates with a middle-ware system build by the Hummingbird Airlines

 

Middle-ware system

 

This is an enterprise application integration (EAI) platform that interfaces with front-end application to different backend applications

 

OP:GetBookingInformation

 

Gets booking information from the given reservation number or passport id

OP:Registration

 

Flight and luggage registration

OP:PrintBoardingPass

 

Print the boarding pass

 

Back-end systems

 

Backend services are provided by partners outside of hummingbird airlines

 

GetBookingInformation

 

This services is provided by the international booking system, the organization that manages the flight reservation for all the airlines

FlightInfo

 

 

This service is provided by the airport, that returns the latest information about the given flight

Luggage Registration

 

 

This service is provided by the airport, that registries luggage.
